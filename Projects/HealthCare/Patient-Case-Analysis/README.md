# Patient Case Analysis

## 📌 Overview
This project analyzes patient case data to provide insights into healthcare operations, including patient distribution, treatment types, and departmental performance. The dashboard highlights key trends across inpatient and outpatient cases.

## 🎯 Objectives
- Analyze patient cases across different categories  
- Compare inpatient and outpatient distributions  
- Identify trends across departments and age groups  
- Support data-driven decision making  

## 📂 Dataset
- **Source:** YouTube tutorial dataset  
- **Rows:** 453,119 records (after data merging)  
- **Columns:** 9 fields (after data processing)  
- **Timeframe:** 2018–2021   

## 📊 Key Metrics
- **Total Patient Cases:** 453,119  
- **Outpatient Cases:** 270,983  
- **Inpatient Cases (including Day Case):** 182,136  

### Case Distribution:
- **Outpatient Rate:** ~60%  
- **Inpatient Rate:** ~40%  

### Age Group Distribution:
- 0–15  
- 16–64  
- 65+  

## 📈 Insights
- **Outpatient cases** (~60%) exceed **Inpatient cases** (~40%), indicating that most cases do not require hospital admission.
- The **16–64 age group** represents the largest share of cases, indicating that most patients are working-age adults.
- **Children (0–15)** represent the smallest share of total cases among all age groups.
